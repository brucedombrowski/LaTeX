\dmsection{Basis for TSU Eligibility}

This software qualifies for License Exception TSU because it meets all requirements under 15 CFR \S~740.13(e):

\begin{itemize}
    \item \textbf{Publicly Available:} Source code is publicly available at \url{\GitHubURL}
    \item \textbf{No Restrictions:} Released under MIT License with no payment or distribution restrictions
    \item \textbf{Standard Cryptography:} Uses standard cryptographic libraries (BouncyCastle, .NET) without custom cryptographic implementations
\end{itemize}

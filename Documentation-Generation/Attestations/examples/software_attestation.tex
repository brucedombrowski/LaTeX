\section*{Checksum Verification}

To verify the integrity of downloaded software:

\begin{verbatim}
# macOS/Linux
shasum -a 256 PdfSigner.exe

# Windows PowerShell
Get-FileHash -Algorithm SHA256 PdfSigner.exe
\end{verbatim}

Compare the output against the checksums documented above.

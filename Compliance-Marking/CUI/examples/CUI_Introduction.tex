\dmsection{Purpose}

This memorandum transmits Controlled Unclassified Information (CUI) cover sheets for documents associated with Project Constellation. Each cover sheet identifies the specific CUI category, dissemination controls, and handling requirements for the attached materials.

% ----------------------------------------------------------------------------
\dmsection{Background}

Project Constellation requires multiple document types with varying CUI classifications. To ensure proper handling and compliance with 32 CFR Part 2002, standardized SF901-format cover sheets have been prepared for each document category.

% ----------------------------------------------------------------------------
\dmsection{Document Contents}

CUI//SP-CTI & Payload Software Interface Control Document \\
\bottomrule
\end{tabular}

% ----------------------------------------------------------------------------
\dmsection{Handling Instructions}

All recipients must adhere to the following requirements:

\begin{itemize}
    \item Review each cover sheet for specific dissemination controls before handling attached documents
    \item Store documents in accordance with 32 CFR Part 2002 and agency policy
    \item Contact the designated Point of Contact (POC) on each cover sheet with handling questions
    \item Destroy documents using agency-approved methods when no longer required
\end{itemize}

% ----------------------------------------------------------------------------
\dmsection{Distribution}

This package is authorized for distribution to U.S. Government agencies and contractors with appropriate need-to-know and security clearances as specified on each individual cover sheet.

\dmsection{Purpose}

This memorandum documents the decision to use Cinzel as the display font for ``CUI'' header text in the SF901 LaTeX template, rather than the Trajan Bold font used in the official GSA template.

% ----------------------------------------------------------------------------
\dmsection{Background}

The official SF901 CUI Cover Sheet template (Standard Form 901, 11-18) uses the \textbf{Trajan Bold} font for the large ``CUI'' header text displayed at the top and bottom of the cover sheet. This font was identified by extracting font metadata from the official PDF template obtained from the National Archives:

\begin{verbatim}
$ pdffonts SF901-Official-Template.pdf
name                          type
ZMPPSD+Trajan-Bold            Type 1C
ZMPPSD+TimesNewRomanPSMT      CID TrueType
ZMPPSD+MinionPro-Regular      Type 1C
\end{verbatim}

Two implementation approaches were considered:

\begin{itemize}
    \item \textbf{Approach A:} Use Trajan Bold (exact match to official template)
    \item \textbf{Approach B:} Use Cinzel (free open-source alternative)
\end{itemize}

% ----------------------------------------------------------------------------
\dmsection{Analysis}

\textbf{Approach A: Trajan Bold (Original Font)}
\begin{itemize}
    \item Exact visual match to official GSA template
    \item Requires purchase (\$35--70) or Adobe Creative Cloud subscription
    \item Not freely redistributable due to commercial license
    \item Adds licensing complexity to the LaTeX template
    \item May not be available on all systems where the template is used
\end{itemize}

\textbf{Approach B: Cinzel (Open Source Alternative)}
\begin{itemize}
    \item Free font directly inspired by Roman inscriptional lettering (same design tradition as Trajan)
    \item Available under SIL Open Font License (OFL) --- permits free use, redistribution, and modification
    \item Available from Google Fonts --- easy installation across Windows, macOS, Linux
    \item Classical Roman capital letterforms with elegant, refined stroke weights
    \item All-caps display styling optimized for titles and headers
    \item Not an exact match, but visually very similar
\end{itemize}

\textbf{Other Alternatives Considered:}
\begin{itemize}
    \item \textbf{Times New Roman Bold:} Pre-installed on most systems, but visually quite different (too heavy, wrong character style)
    \item \textbf{Times New Roman Regular:} Pre-installed with lighter weight, but different letterform style and not a display font
\end{itemize}

% ----------------------------------------------------------------------------
\dmsection{Decision}

\textbf{Approved:} Approach B --- Use Cinzel as the CUI header font.

\textbf{Rationale:}
\begin{itemize}
    \item Eliminates licensing costs and restrictions on template distribution
    \item Maintains visual consistency with official SF901 appearance
    \item Widely available and well-maintained font
    \item Easy installation via Google Fonts
    \item The visual difference from Trajan is minimal and acceptable for this use case
\end{itemize}

\textbf{Implementation:}
\begin{itemize}
    \item Install Cinzel font on target systems (documented in SF901.tex)
    \item Define \texttt{\textbackslash cuifont} in SF901.tex using fontspec package
    \item Apply \texttt{\textbackslash cuifont} to CUI header text nodes
    \item Document font installation in build instructions
\end{itemize}

\textbf{Font Installation (macOS):}
\begin{verbatim}
curl -L -o /tmp/cinzel.zip \
  "https://gwfh.mranftl.com/api/fonts/cinzel?download=zip\
&subsets=latin&variants=regular"
unzip -o /tmp/cinzel.zip -d ~/Library/Fonts/
\end{verbatim}
